### **Quick Installation with uv**
```bash
uv init japanese-ocr-rag --python 3.12
cd japanese-ocr-rag
uv lock
uv sync
```


Both files include the **critical note**: Use PaddlePaddle **2.6.0**, NOT 3.0.0 (NCCL conflict with PyTorch 2.7).
